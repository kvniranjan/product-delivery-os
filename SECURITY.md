# Security and Privacy

Do not commit:

- Client data
- Company data
- Customer data
- Credentials or API keys
- Internal meeting transcripts
- Production data
- Confidential architecture details
- Meeting recordings

Use `workspace.example/` as a template only. Copy it to `workspace/` for real project work. The `workspace/` folder is gitignored by default.

AI-generated artifacts must be reviewed by a human before delivery. Regulatory, legal, financial, medical, or compliance-sensitive outputs require expert review.

## Reporting Issues

If you find a security or privacy issue, report it responsibly through the repository maintainers. Do not publish sensitive details in a public issue.
