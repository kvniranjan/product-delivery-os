# Core Banking Glossary

| Term | Meaning |
|---|---|
| Core banking platform | Primary system of record for customer accounts, balances, loans, and transactions. |
| Earmark/Hold | Restriction placed on funds or account activity pending settlement, review, or release. |
| GL | General Ledger used for accounting postings and reconciliation. |
| EOD | End-of-day processing, batch updates, posting, reconciliation, and reporting. |
| Middleware | Integration layer that transforms, routes, or orchestrates messages between systems. |
| Payment rail | Network or method used to move money. |
