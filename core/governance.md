# Governance

- Do not commit confidential company data.
- Do not commit customer data.
- Do not commit credentials or API keys.
- Use `workspace.example/` as a template only.
- Actual user work should live in `workspace/` and be gitignored.
- AI outputs should be reviewed by a human before delivery.
- Regulatory, legal, financial, medical, or compliance-sensitive outputs require expert review.
- Decision logs should clearly separate recommendation from approval.
- AI-generated artifacts should identify assumptions and open questions.
