# Codex Adapter

Use this adapter when applying Product Delivery OS inside a Codex-enabled repository.

Codex uses `AGENTS.md` for repository instructions and skills as reusable playbooks. The source of truth remains the AI-agnostic files in `core/`, `role-packs/`, `workflows/`, `templates/`, and `prompt-packs/`.

Copy `AGENTS.md.template` into your working repository as `AGENTS.md`, then add only the skills you need.
